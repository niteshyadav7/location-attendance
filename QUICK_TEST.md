# 📱 Quick Testing Steps

## 🎯 **Simple 5-Minute Test**

### **What You Need:**
- 2 devices (or 1 device + 1 emulator)
- 1 admin account
- 1 user account

---

## ✅ **Testing Steps**

### **Device 1 (Admin):**
```
1. Open app
2. Login as admin
3. Allow notifications
4. Minimize app (keep it running!)
```

### **Device 2 (User):**
```
1. Open app
2. Login as user
3. Click "Check In"
```

### **Device 1 (Admin):**
```
🔔 Notification appears!
"✅ User Checked In"
```

---

## 🎉 **That's It!**

If you see the notification, **it's working!**

---

## 🧪 **Test All Actions:**

On **User Device**, try:
- ✅ Check In
- ☕ Take Break
- 💼 Resume Work
- 👋 Check Out

**Admin gets notification for each!** 🔔

---

## ⚠️ **Important:**

**Admin app must be running** (background is OK)

Don't force-close the admin app!
